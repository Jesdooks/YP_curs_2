Система управления хлебопекарней
=============================

Этот документ описывает систему управления хлебопекарней, созданную на Python с использованием Tkinter для интерфейса и SQLite для базы данных. Система предназначена для локального использования в хлебопекарне, поддерживает роли администратора, клиента.

Учетные данные для входа
-----------------------

Для доступа к системе используйте следующие логины и пароли:

- Администратор: логин "0", пароль "0"


- Убедитесь, что файлы изображений и шрифтов находятся в указанных папках.
- При проблемах с доступом к базе данных (ошибка "database is locked") закройте другие экземпляры приложения.
- Система рассчитана на локальное использование в одной пекарне.
